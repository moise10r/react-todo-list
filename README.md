- ReactJS

## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

```bash
git clone git@github.com:moise10r/react-todo-list.git
cd ./react-todo-list
```

### Install

```bash
npm install
```

### To Run it

```bash
npm start
```

### Build

```bash
npm run build
```
